## Installation

Install ProtoCore with the install.sh script:

```bash
  sudo apt install git
  git clone https://github.com/DrekkTheProto/ProtoCore.git
  cd ProtoCore
  ./install.sh
```
Then reboot your Raspberry PI

To run the code:
```bash
  cd ProtoCore
  sudo python3 Proto.py
